`@automapper/pojos` is one of the official plugins of `@automapper/*` monorepo. `@automapper/pojos` works with plain objects along with their Interfaces/Types.

## Installation

```bash
npm i @automapper/pojos
# or with yarn
yarn add @automapper/pojos
```
